Currency readme

What it is?
This application compares different currency rates. It reads from a data file called rates.csv and presents the data in different ways. In order to give correct results make sure the datepickers have values between 2013-11-24 and 2015-11-30.

Documentation
The documentation is in currency/doc. Open index.html with your favourite browser.
